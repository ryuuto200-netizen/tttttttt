GitHub Pages公開用のリポジトリです。

## 最後の作業

1. 完成版HTMLのファイル名を `index.html` に変更する。
2. このリポジトリ直下にある既存の `index.html` を、完成版で置き換える。
3. GitHubの `Settings` → `Pages` を開く。
4. `Build and deployment` の `Source` を `Deploy from a branch` にする。
5. Branchを `main`、フォルダーを `/(root)` にして `Save` を押す。
